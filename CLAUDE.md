This repo is a new app for a light auditing platform. Users will use a PWA to audit a location and then load the data into a web platform which they will then use to make recommendations for sales/installation.

# Project structure
This is a django app and the audit app is the main app in the project.
There is a folder - html_app at the root that is a huge html file. This is the "audit" tool where users will use locally as a PWA and upload data. This needs to be fixed up at some point but is a pre-existing piece of the puzzle. For now it should be used as a north star for our data model.
